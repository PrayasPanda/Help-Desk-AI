# Shared Drive and Folder Access

Access to departmental shared drives (Finance, HR, Legal, Engineering) is granted per-folder and requires approval from the folder's owner, listed in the drive catalog at drives.internal.

To request access: open drives.internal, find the folder, click "Request Access", pick a duration (30, 90, or 365 days) and state a business reason. The owner is notified immediately; most requests are decided within one working day. Access expires automatically at the end of the chosen duration and can be renewed from the same page.

The Finance drive is restricted: it additionally requires sign-off from the Finance Director, which can add 2-3 working days. Plan ahead for month-end.

If you already have access but see "permission denied", your grant has likely expired — check "My Access" on drives.internal before raising a ticket. When you do raise one, include the exact folder path and the error message.
