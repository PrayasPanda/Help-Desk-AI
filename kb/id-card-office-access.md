# ID Card and Office Access

Your ID card opens the main entrance, your floor, and common areas between 06:00 and 22:00 on working days. Weekend or after-hours access needs a request from your manager to Admin (admin.internal → Access Requests).

Lost or damaged card: report it immediately on admin.internal so the old card is deactivated, then collect a replacement from the Admin desk on the ground floor (bring a government ID). The first replacement in a calendar year is free; subsequent ones cost $10, billed to payroll.

Forgot your card at home? The Admin desk issues a one-day visitor pass — no manager approval needed, maximum three times per month.

Parking access uses the same card. If the card opens doors but not the parking barrier, that is a separate permission — request "Parking" on the same Access Requests page. Card readers showing a solid red light are faulty; report the reader location in an Admin ticket.
